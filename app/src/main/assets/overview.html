The Linuxdroid:Boot add-on provides functionality to run programs under Linuxdroid when the device boots up.

Usage

  • Start the Linuxdroid:Boot app once to allow the system to start the app at boot time.
  • Create the ~/.linuxdroid/boot/ directory.
  • Put scripts you want to execute inside the ~/.linuxdroid/boot/ directory.
  • If there are multiple files, they will be executed in a sorted order.
  • Note that you may want to run linuxdroid-wake-lock as first thing if you want to ensure that the device is prevented from sleeping.

Example

To start an sshd server and prevent the device from sleeping at boot, create a file at ~/.linuxdroid/boot/start-sshd containing the two lines

    linuxdroid-wake-lock
    sshd
